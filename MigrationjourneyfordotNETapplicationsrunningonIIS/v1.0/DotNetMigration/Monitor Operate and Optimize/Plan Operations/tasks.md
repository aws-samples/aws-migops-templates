
# Module: Plan Operations
## Task 1: Check on operational roles and responsibilities
#### Description
Ensure that the operational roles and responsibilities defined in the Mobilize phase are correct and current. Use defined procedures to run operations effectively.
#### Tools
Create a RACI or RASCI matrix for a cloud operating model
#### Acceptance Criteria
• Check and update the operational roles and responsibilities RACI matrix as needed. 

## Task 2: Create playbooks and troubleshooting documents
#### Description
Once the application migration is complete in AWS, ensure that the team creates playbooks and troubleshooting/incident management documents. These playbooks and documents will help operation teams ensure that the environment is stable and available at all times.
#### Acceptance Criteria

• Created operational runbooks
• Created troubleshooting documents
• Created plans for application updates
